# Logic Circuit Simulator

Interactive logic gate simulator built with React Flow, Express.js, and MongoDB.

## Tech Stack
- **Frontend:** React (Vite), React Flow, Zustand
- **Backend:** Node.js, Express.js, Mongoose (3-Layered Architecture)
- **Infrastructure:** Docker, Docker Compose

## How to run locally

1. Make sure you have Docker and Docker Compose installed.
2. Clone the repository.
3. Run the following command in the root directory:

```bash
docker-compose up --build
```

4. Open `http://localhost:5173` for the frontend.
5. The backend API runs on `http://localhost:3000`.
